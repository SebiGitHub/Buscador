# Buscador (Android)

## Qué es
App Android para practicar un buscador: entrada de texto, filtrado de resultados y presentación en lista/detalle.

## Stack
- Kotlin
- Android Studio
- UI: RecyclerView
- Datos: API / JSON

## Features
- Búsqueda por texto con filtrado en tiempo real
- Lista de resultados + pantalla de detalle
- Manejo de estados: vacío, cargando, sin resultados

## Cómo ejecutar
1. Clona el repo
2. Abre en Android Studio
3. Ejecuta en emulador o dispositivo
4. (Si usa API) configura endpoint/keys

## Qué aprendí
- UX de búsqueda (debounce, estados, feedback)
- Filtrado eficiente y actualización de listas
- Estructurar datos y UI de forma limpia
